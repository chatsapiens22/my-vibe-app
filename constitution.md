# prj_0618 — Constitution

## 프로젝트 목표
- 가공의 후원자 데이터 200개 생성 (이름, 연락처, 이메일, 후원 유형, 후원 금액, 생일, 납부 방식, 가입일, 상태, 관심분야 포함)
- 후원자 데이터를 조회하고 필터링할 수 있는 MVP 웹 대시보드 구축을 위한 기초 데이터 제공
- 복지 기관 담당자가 후원자 현황을 시각적으로 파악하고 생일 후원자를 간편하게 관리할 수 있는 모던한 대시보드 웹 애플리케이션 개발

## 불변 규칙
- 원본 데이터/로그 파일은 절대 수정하지 않는다 (읽기 전용)
- API 키/비밀키는 환경 변수로만 관리한다
- 사용자 확인 없이 파일을 삭제하지 않는다
- 생성되는 모든 데이터는 가공의 데이터(가짜 정보)이며 실제 개인정보를 노출하지 않는다.

## 성공 기준
- 200개의 후원자 데이터가 JSON 및 CSV 파일로 정확히 생성됨.
- 각 데이터 레코드는 생일(birthday) 필드를 포함하며, 날짜가 YYYY-MM-DD 형식으로 유효함.
- `index.html`, `style.css`, `app.js` 기반의 대시보드 웹 앱이 정상 작동함.
- 요약 지표 집계, 3종 통계 차트(관심분야, 연령대, 납부방식), 검색 및 다중 필터링, 오늘/이번 달 생일자 자동 감지 및 가상 축하 발송 토글 기능이 에러 없이 완벽히 동작함.
